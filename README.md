# 🌊 Chaos Snake: Underwater Sector

Um jogo de sobrevivência rítmica e reflexos rápidos ambientado nas profundezas do oceano. Esta é uma versão independente e aprimorada da franquia "Chaos", focada em jogabilidade fluida para dispositivos móveis e navegadores.

## 🎮 O Jogo
Em **Chaos Snake: Underwater Sector**, o jogador controla uma sonda subaquática que deve limpar setores infestados por criaturas abissais.

### Funcionalidades Principais:
* **Zonas de Pressão:** Níveis especiais com limite de tempo (60 segundos) para aumentar a adrenalina.
* **Sistema de Cristais:** Colete 3 corais mágicos para quebrar as barreiras de proteção dos inimigos.
* **Power-ups:** Utilize as *Wild Cards* (✨) para entrar em modo de fúria e eliminar obstáculos.
* **Dificuldade Progressiva:** Inimigos como sentinelas, medusas e polvos tornam-se mais agressivos a cada zona.

## 🚀 Tecnologias Utilizadas
* **HTML5 & CSS3:** Interface responsiva com efeitos de iluminação neon e movimento de água.
* **JavaScript (Canvas API):** Motor de jogo customizado para renderização de partículas e lógica de colisão.
* **IndexedDB:** Sistema de salvamento local para manter o recorde de profundidade do jogador.

## 🛠️ Como Jogar
O jogo foi otimizado para **dispositivos móveis** (touch) e futuramente para **Desktop**
